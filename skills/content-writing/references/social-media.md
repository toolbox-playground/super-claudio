# Social Media Content

## TikTok / Instagram Reels — Hook Formulas

The first 2-3 seconds determine if viewers watch or scroll. Proven hook patterns:

**Question hook:** "Did you know [surprising fact]?"
**Controversy hook:** "Nobody talks about [uncomfortable truth]..."
**How-to hook:** "Here's how I [result] in [timeframe]..."
**Story hook:** "I almost [failure] until I discovered..."
**Challenge hook:** "Try this for 7 days and see what happens..."

### TikTok Caption Formula
```
[Hook sentence — repeats the video hook]
[1-2 lines of value or context]
[CTA: Follow for more / Link in bio / Comment X]
[3-5 hashtags: 1 niche + 2 medium + 1-2 broad]
```

### Video Script Structure (60 seconds)
- 0-3s: Hook (problem or bold claim)
- 3-15s: Context (why this matters to YOU)
- 15-45s: Value (the actual tip/demo/story)
- 45-55s: Proof or result
- 55-60s: CTA

## Instagram — Feed Post Captions

**Long caption formula (saves and shares):**
- Line 1: Hook (stops the scroll)
- Lines 2-5: Story or value (hidden behind "more")
- Lines 6-8: Key takeaway or list
- Final line: CTA + hashtags

**Short caption formula (engagement):**
- One punchy line + CTA
- 5-10 hashtags in first comment

### Hashtag Strategy
- 3-5 niche hashtags (under 500k posts) — for reach in your community
- 3-5 medium hashtags (500k–5M posts) — balance of reach and competition
- 2-3 broad hashtags (5M+ posts) — small chance, high upside

## X/Twitter Threads

**Thread structure:**
- Tweet 1: Bold claim or promise ("Here's how I X in Y days:")
- Tweets 2-8: One insight per tweet, each standalone
- Tweet 9: Summary of key points
- Tweet 10: CTA (follow, newsletter, product)

**Rules:**
- Each tweet must be valuable alone (people share individual tweets)
- Use numbers: "3 things", "7 steps", "1 mistake"
- End each tweet with a hook to the next (leave them wanting more)

## Prompt Templates for Claude

Generate TikTok hooks:
```
Write 5 TikTok hook variations for a video about [topic].
Audience: [describe]. Format: punchy, under 10 words each.
```

Generate Instagram carousel:
```
Create a 7-slide Instagram carousel about [topic].
Slide 1: Hook. Slides 2-6: One point each. Slide 7: CTA.
Keep each slide under 30 words. Tone: [casual/professional].
```

Generate X thread:
```
Write a 10-tweet thread about [topic] for [audience].
Tweet 1 should be a bold claim. Each tweet standalone value.
Last tweet: CTA to [action].
```
