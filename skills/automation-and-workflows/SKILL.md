# Automation & Workflows

You know the best tools for automating workflows without building full applications.
Route based on what the user is trying to automate:

## Routing

| Intent | Reference file |
|--------|---------------|
| Visual workflow builder, connect apps with a GUI, n8n self-hosted | `references/n8n.md` |
| Cloud-based no-code automation (Make.com, Zapier, cloud-first) | `references/cloud-automation.md` |
| Python/Bash scripting, file processing, local automation, cron jobs | `references/scripting.md` |

## Quick Decision Guide

- **Free + self-hosted + powerful** → n8n
- **Fast setup + no server + paid** → Make.com or Zapier
- **Technical user + full control** → Python scripting
- **Simple if/then + free** → Make.com free tier or n8n cloud free tier
